
## Essential Tasks

When one side is entering a message (talking) the other side is not able to type anything.

The indicators show if one side is talking, in which case the other side is busy, or if both are open.

Pressing enter sends the message to the other side.

Messages are displayed in the message panel on the receiving side.

The arrow in the middle confirms the direction that talking is happening or shows a double arrow if no side is talking.
